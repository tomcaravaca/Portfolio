  • Notifications Telegram via bot personnalisé
  • Synchronisation des données vers GitHub (flowly-data.json)
  • Installable sur iPhone depuis Safari (PWA)
  • Score "Flowly" calculé sur les statistiques d'utilisation
HTML CSS JS Python DSM Stage Médiaconcept

Dashboard NAS

Développé pendant mon stage de 6 semaines chez Médiaconcept, une agence d'informatique à la Rochefoucault. La mission : créer une interface de centralisation pour un NAS Synology utilisé en interne.

Contrainte principale : zéro dépendance externe. Tout devait tourner en HTML/CSS/JS pur, sans librairie, pour rester léger et maintenable.

  • Affichage en temps réel des données système (CPU, RAM, stockage)
  • Lecture des données via l'API REST du NAS Synology
  • Interface responsive utilisable depuis n'importe quel poste du réseau
  • Aucune dépendance, fichier HTML autonome
  • Mise à jour automatique des données toutes les 10 secondes

Pas de lien public disponible, interface interne à l'entreprise.
